Task Management
1
➕ Add Tasks
Click the + Task button in the header. You can add Title, Category, Priority (High/Medium/Low), Due Date, Notes, Tags, and even Subtasks!
2
☀️ Switch Views
In the sidebar, you'll find views for Today (tasks for today), All Tasks (all pending tasks), and Completed (finished tasks).
3
🔍 Search & Filter
Search for tasks by name or tags on the dashboard. You can also filter or sort them by priority and date!
💡
Pro Tip: Add multiple subtasks to a single task — completing each subtask earns you extra XP!

Pomodoro & Study Mode
⏱
Pomodoro Timer
Sidebar
You'll find the POMODORO section in the sidebar.

• ▶ Start — Begin a focus session (25 min)
• ↺ — Reset timer
• edit — Customize Focus/Break intervals

Earn +50 XP for completing a session!
📚
Study Mode
Header
Click the 📚 Study button in the header.

• Fullscreen distraction-free timer
• Add a session note (e.g. "Math Ch.3")
• 💾 Save Session — Log your time
• ✕ Discard — Cancel session
⚡
Pomodoro Technique: Work for 25 min, take a 5 min break. After 4 sessions, take a longer 15-30 min break. It's a scientifically proven productivity method!
