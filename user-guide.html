Task Management
1
➕ Add a Task
Click the + Task button in the top header. Fill in the title, category, priority (High / Medium / Low), due date, notes, tags, and even subtasks.
2
☀️ Switch Views
Use the sidebar to navigate between Today (tasks due today), All Tasks (all pending tasks), and Completed (finished tasks).
3
🔍 Search & Filter
Search tasks by name or tags using the search bar. Filter by priority and sort by Date, Priority, or Name using the dropdowns.
4
✅ Complete a Task
Click the checkbox next to any task to mark it complete — you'll earn XP instantly! Click the task card to view full details.
💡
Pro Tip: Add multiple subtasks inside a single task — each subtask completion earns you bonus XP!
🏷️
02 / 06
Categories
⚙️
Manage Categories
Sidebar
Click the gear icon ⚙ next to "CATEGORIES" in the sidebar. Create new categories, pick an emoji, and choose a color to identify them easily.
🎨
Custom Colors & Emojis
Custom
Assign a unique emoji and color to each category for quick visual recognition. Example: 📚 Study (blue), 💼 Work (green), 🏃 Health (orange).
📊
Category Stats
Stats
Open the Stats view to see how many tasks belong to each category and what your completion rate is per category.
🎯
Suggestion: Start with categories like Study, Work, Personal, and Health — this helps you track exactly where your time goes.
🔄
03 / 06
Habit Tracker
1
🔄 Open Habit Tracker
Click Habit Tracker in the sidebar to open your daily habits dashboard.
2
➕ Add Daily Habits
Add habits you want to build — such as Exercise, Reading, or Meditation. Start small with 2–3 habits per day for best results.
3
🔥 Build Your Streak
Mark each habit as done every day to grow your streak. A longer streak means more XP and stronger motivation to keep going!
🔥
Challenge: Try building a 7-day streak! Research shows repeating an action for 21 days turns it into a lasting habit.
⏱
04 / 06
Pomodoro & Study Mode
⏱
Pomodoro Timer
Sidebar
Find the POMODORO section in the sidebar:

• ▶ Start — Begin a 25-minute focus session
• ↺ — Reset the timer
• edit — Customize your Focus and Break durations

Completing each session earns you +50 XP!
📚
Study Mode
Header
Click the 📚 Study button in the header to launch a fullscreen, distraction-free timer:

• Add a session note (e.g. "Math Chapter 3")
• Select a category
• 💾 Save Session — Log your study time
• ✕ Discard — Cancel without saving
⚡
Pomodoro Technique: Work for 25 minutes → take a 5-minute break. After 4 sessions, take a longer 15–30 minute break. Scientifically proven to boost focus and reduce burnout!

Install as Mobile App
PWA
