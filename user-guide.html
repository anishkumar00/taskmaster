Task Management
1
➕ Task Add Karo
Header mein + Task button click karo. Title, Category, Priority (High/Medium/Low), Due Date, Notes, Tags aur Subtasks add kar sakte ho!
2
☀️ Views Switch Karo
Sidebar mein Today (aaj ke tasks), All Tasks (saare pending tasks), aur Completed (done tasks) views hain.
3
🔍 Search & Filter
Dashboard pe tasks ko name ya tags se search karo. Priority aur Date se filter/sort bhi kar sakte ho!
💡
Pro Tip: Ek task mein multiple subtasks add karo — har subtask complete karne par extra XP milta hai!

Pomodoro & Study Mode
⏱
Pomodoro Timer
Sidebar
Sidebar mein POMODORO section hai.

• ▶ Start — Focus session shuru karo (25 min)
• ↺ — Reset karo
• edit — Focus/Break time customize karo

Har session complete karne par +50 XP milta hai!
📚
Study Mode
Header
Header mein 📚 Study button click karo.

• Fullscreen distraction-free timer
• Session note add karo (e.g. "Math Ch.3")
• 💾 Save Session — Time log karo
• ✕ Discard — Cancel karo
⚡
Pomodoro Technique: 25 min kaam karo, 5 min break lo. 4 sessions ke baad 15-30 min ka lamba break lo. Yeh scientifically proven productivity technique hai!
